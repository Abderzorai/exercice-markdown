Markdown

Présentation

Le Markdown est un langage de balisage crée en 2004 par John Gruber et Aaron Swartz. C'est un langage qui se veut simple à lire et à utiliser.

Le Markdown a pour but de simplifier l'écriture sur internet. Il n'est pas concurrent avec un langage comme HTML car il reste très limité. Il est
plus "complémentaire". D'ailleurs, vous pouvez utiliser le Markdown dans une page HTML.

Point positifs du MarkDown

  • Le poids des documents rédigés en MarkDown est léger.
  • Il est facile de convertir un document édité en Markdown à un document HTML.

Point négatif du MarkDown

  • Pour introduire une image, vous devez obligatoirement passer par un lien. Il faut donc héberger son image sur un serveur

Usage

  • Le MarkDown n’exige pas de connaissance en informatique pour être utilisé, ce qui permet à tout le monde de s’en servir.
